# 📒 Contact Book

A command-line **Contact Management System** built with Python that lets users store, search, update, and delete contacts — with all data saved persistently using JSON file storage.

---

## 🚀 Features

- ➕ Add new contacts (name, phone, email)
- 🔍 Search contacts by name
- 📋 View all saved contacts in a formatted list
- ✏️ Update existing contact details
- 🗑️ Delete contacts
- 💾 Persistent storage using JSON — data survives after the program closes

---

## 🛠️ Technologies Used

- **Language:** Python 3
- **Concepts:** Functions, File Handling, JSON, CRUD Operations, OOP Logic
- **Storage:** JSON (data persists between sessions)

---

## 📌 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/priyanshupatel-tech/Contact-Book-Python.git
   cd Contact-Book-Python
   ```

2. Run the program:
   ```bash
   python Project5.py
   ```

3. Follow the on-screen menu to manage your contacts.
